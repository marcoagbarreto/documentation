Installing Anaconda on Windows¶

Anaconda is a frontend enviroment for Python language. It is necesary for setting up your workspace. Installation packages are available for Windows, Mac and Linux. We are going to go through the Windows installation.

  1. Using a Web browser, navigate to the Anaconda’s Individual Edition page. Then, go to the download section.

  2. Click on your preferred installer. In our case we are choosing the 64-Bit Windows installer since we are using a 64-Bit computer.

  3. Save the .EXE file

  4. Go to your download folder and open the Anaconda .EXE file

  5. Go through the installation process as an usual Windows installation.
